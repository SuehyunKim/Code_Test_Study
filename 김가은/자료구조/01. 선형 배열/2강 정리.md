## 리스트(배열) 연산

```python
L = [1, 2, 3, 4]
```

### 원소 덧붙이기

```python
L.append(5)
L = [1, 2, 3, 4, 5]
```

### 끝에서 꺼내기

```python
L.pop()
return 4
L = [1, 2, 3]
```

### 원소 삽입

```python
L.insert(idx, 원소)
```

### 객체 삭제

```python
del L[idx]
```

<hr>

### append와 pop의 시간복잡도: O(1)

-> 리스트 맨 끝에 바로 접근

### insert와 del의 시간복잡도: O(n) (n:원소개수)

-> 삽입하려는 위치 이후의 원소들을 모두 한칸씩 뒤로 옮김
-> 삭제하려는 위치 이후의 원소들을 모두 한칸씩 앞으로 옮김
